## ?? Technical Reference

### Files Involved

#### Frontend
```
Views/Customer/Index.cshtml          ? Advanced grid
Views/Product/Index.cshtml           ? Simple table
Views/Cart/Index.cshtml              ? Simple table
Views/Shared/_Layout.cshtml          ? Main layout
```

#### Backend
```
Controllers/CustomerController.cs    ? Grid logic
Data/ApplicationDbContext.cs         ? Database context
Models/Customer.cs                   ? Customer model
```

#### Styling
```
wwwroot/css/customer-management.css  ? Grid styling
wwwroot/css/fashion-shop.css         ? Global styling
```

See: **CUSTOMER_MANAGEMENT_SUMMARY.md** > "File Structure"

### Database

#### Tables
- `Customers` - Customer information
- `Products` - Product catalog
- `Carts` - Shopping carts
- `CartItems` - Items in carts
- `Invoices` - Orders/Invoices
- `InvoiceDetails` - Invoice items
- `Reviews` - Product reviews

#### Relationships
- 1 Customer ? Many Carts/Invoices/Reviews
- 1 Product ? Many CartItems/Reviews/InvoiceDetails
- 1 Cart ? Many CartItems
- 1 Invoice ? Many InvoiceDetails

See: **CUSTOMER_MANAGEMENT_SUMMARY.md** > "Database"
