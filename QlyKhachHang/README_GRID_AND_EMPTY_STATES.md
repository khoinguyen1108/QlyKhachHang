## ?? Key Concepts

### Empty State Logic
```
IF (Model has data)
{
    Show table with data
}
ELSE
{
    Show friendly "no data" message
}
```

### Automatic Display
- ? No data ? "Hi?n không có d? li?u" message
- ? Has data ? Table automatically displays
