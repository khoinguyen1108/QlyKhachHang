## ?? Interaction Flow

### Search Flow
```
User enters "John"
        ?
Click [?? Tìm ki?m]
        ?
Browser: /Customer/Index?searchTerm=John
        ?
Server: Filter by:
  - CustomerName LIKE %John%
  - Phone LIKE %John%
  - Email LIKE %John%
  - City LIKE %John%
        ?
Display matching results (1-10 per page)
        ?
Show: "Trang 1 / 3" (if 30 results found)
```

### Filter Flow
```
Select Status: "Active"
        ?
Select Sort By: "CreatedDate"
        ?
Select Sort Order: "Desc"
        ?
Click [?? Tìm ki?m]
        ?
URL: /Customer/Index?statusFilter=Active&sortBy=CreatedDate&sortOrder=desc
        ?
Server: Filter WHERE Status='Active' ORDER BY CreatedDate DESC
        ?
Display sorted active customers
```

### Sort Flow
```
User sees column: "Tên Khách Hàng ?"
        ?
Click on header
        ?
URL changes: ?sortBy=CustomerName&sortOrder=desc
        ?
Column shows: "Tên Khách Hàng ?"
        ?
Data sorted Z ? A (Descending)
        ?
Click again
        ?
Column shows: "Tên Khách Hàng ?"
        ?
Data sorted A ? Z (Ascending)
```

### Pagination Flow
```
View page 1 (default)
        ?
User clicks page [3]
        ?
URL: /Customer/Index?page=3
        ?
Server: Skip 20 records, Take 10 records (PAGE_SIZE=10)
        ?
Display customers 21-30
        ?
Show: "Trang 3 / 125"
        ?
Highlight [3] button
```
